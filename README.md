# github-user-repos-search
Demo React app: searching github repos by username.
Config app variables are located in .env file in root.

## Install

Install with npm:

```bash
npm install
```

Install with yarn:

```bash
yarn install
```

## Run

Development:

```bash
yarn run dev
```

Build:

```bash
yarn run build
```

Build version with server:

```bash
yarn run build-server
```
